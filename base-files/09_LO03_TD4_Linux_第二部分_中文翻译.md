# LO03  
## Linux 第 4 次习题课（第二部分）

### 正则表达式速查表

- `^`：行首
- `$`：行尾
- `.`：任意单个字符
- `*`：重复 0 次或多次
- `+`：重复 1 次或多次（需配合 `-E`）
- `?`：重复 0 次或 1 次（需配合 `-E`）
- `{n}`：恰好重复 n 次
- `{n,}`：至少重复 n 次
- `{n,m}`：重复 n 到 m 次
- `[]`：字符类
- `[^]`：字符类取反
- `|`：逻辑或（需配合 `-E`）
- `()`：分组（需配合 `-E`）
- `\`：转义
- `\b`：单词边界
- `[0-9]`：数字
- `[a-z]`：小写字母
- `[A-Z]`：大写字母

### 练习 1

给定如下内容：

```text
asterix ;
20 tintin / haddock ;
20 tif / tondu ;
30 theodore poussin ;
40 spirit ;
30
```

请完成以下修改操作（各小问彼此独立处理）：

1. 删除包含 `30` 的行。  
2. 将小写字母替换为大写字母。  
3. 查找所有**第二个字母是 `s`** 的单词。  
4. 查找所有以 `t`、`s` 或 `h` 开头的单词。  
5. 查找所有**不包含数字**的行。  
6. 用什么命令可以提取**第 4 行到第 6 行**中每一行的**第一个字符**？

### 练习 2

1. 写一条命令，在文件中查找所有包含**同一个字符连续出现 3 次或以上**的字符串，例如 `"aaabbb"` 或 `"111"`。  
2. 写一条命令，查找包含所有元音字母（`a, e, i, o, u`）且每个至少出现一次的句子。  
3. 写一条命令，查找文件中**恰好包含 3 个单词**的行。  
4. 检测长度**至少为 10 个字符**的单词。

### 练习 3

编写一个 Bash 脚本 `substitution`：  
将**第一个参数**传入的字符串的所有出现位置，替换为**第二个参数**传入的字符串，替换操作在**第三个参数**指定的文件中进行。  
结果需要保存回该文件中。

### 练习 4

如何查找一个文件中**包含同一个单词出现 3 次**的所有行？

### 练习 5

考虑一个文本文件，其内容由**大写字母、小写字母和空格**组成。  
编写一个 Bash 脚本，判断该文本是否是一个 **“grobalogramme”**。

这里的 **grobalogramme** 指的是：  
文本中的每个单词都必须满足：

- 以同一个**大写字母**开头和结尾；
- 在这两个首尾大写字母之间，**不能再出现大写字母**。

例如，下面这段文本就是一个 **grobalogramme**：

```text
BoB EmpruntE TemporairemenT A RogeR SeS
SoulierS
```

### 练习 6

考虑一个文本文件 `allo.txt`，其中每一行对应公司的一名员工，字段之间用字符 `:` 分隔。  
这些字段分别表示：**名、姓、电话号码、邮箱地址**。

示例：

```text
Paul:Dupont :0144567799 :dupont@club.fr
Julie:Dalbanot:0469857468:julie@wanadoo.fr
George:Martin :0569874587:martin@free.fr
```

1. 给出一条命令，将文件 `allo.txt` 转换为：
   - **交换姓和名的位置**
   - 并将分隔符 `:` 替换为竖线 `|`

2. 给出一条命令，将 `allo.txt` 转换为如下展示形式，即每个员工的信息按 **3 行**输出，例如：

```text
Paul Dupont
Tél : 0144567799
Email : dupont@club.fr
```
